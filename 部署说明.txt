# 电商消费大数据可视化分析系统部署说明
在本地环境部署“电商消费大数据可视化分析系统”。系统采用前后端分离架构：
- **后端**：Java + Spring Boot + Maven + MySQL


---

## 2. 环境准备
### 2.1 获取源码
后端项目（含 pom.xml）
前端项目（含 template）

### 2.2 工具清单
- JDK 1.8 或 11
- Maven 3.6+
- MySQL 8
- IntelliJ IDEA

## 3. 后端部署
### 3.1 安装与配置 JDK
1. 下载并安装 JDK（例如安装到 `C:\Program Files\Java\jdk1.8.0_191`）。
2. 配置环境变量：
   - 新建系统变量 `JAVA_HOME`，值为 JDK 安装路径。
   - 编辑系统变量 `Path`，新增 `%JAVA_HOME%\bin`。
3. 验证：打开命令提示符，执行 `java -version`，显示版本信息即成功。

### 3.2 安装与配置 Maven（使用阿里云镜像）
1. 下载 Maven 二进制包（如 `apache-maven-3.6.3-bin.zip`），解压到无中文空格的目录（例如 `D:\apache-maven`）。
2. 配置环境变量：
   - 新建 `MAVEN_HOME`，值为解压路径。
   - 编辑 `Path`，新增 `%MAVEN_HOME%\bin`。
3. 配置阿里云镜像（加速依赖下载）：
   - 打开 Maven 安装目录下的 `conf/settings.xml` 文件。
   - 在 `<mirrors>` 标签内添加：
     ```xml
     <mirror>
         <id>aliyunmaven</id>
         <mirrorOf>central</mirrorOf>
         <name>阿里云公共仓库</name>
         <url>https://maven.aliyun.com/repository/central</url>
     </mirror>
	 
MySQL 安装与数据库导入
安装 MySQL：使用集成环境一键安装 MySQL，端口（默认 3306）、用户名（默认 root）和密码。	 

在 IDEA 中配置 JDK
打开 IntelliJ IDEA，点击 File → Project Structure（或 Ctrl+Shift+Alt+S）。
在左侧选择 SDKs，点击 + → Add JDK，选择 JDK 安装目录，点击 OK。

在 IDEA 中配置 Maven
点击 File → Settings → Build, Execution, Deployment → Build Tools → Maven。
将 Maven home path 指向的 Maven 安装目录。
勾选 Override 并选择 User settings file 为你修改过的 settings.xml 文件。
点击 Apply 保存。


启动前端
在终端执行启动命令（具体命令查看的 scripts 部分）：
npm run serve
